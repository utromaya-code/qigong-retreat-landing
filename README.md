# Qigong Retreat Landing

Лендинг ретрита по цигун «Вкус жизни» (21-23 февраля 2026).

## Структура
- `index.html` — основная страница
- `styles.css` — стили
- `script.js` — интерактив
- `privacy.html` — шаблон политики
- `offer.html` — шаблон оферты
- `assets/images/` — изображения

## Сайт в сети
**https://utromaya-code.github.io/qigong-retreat-landing/**

## Запуск локально
```bash
cd qigong-retreat-landing
python3 -m http.server 8080
```
Откройте `http://localhost:8080`.

## Реализовано
- Полная структура секций по ТЗ
- Адаптивная верстка (mobile-first)
- Sticky header + мобильное меню
- Плавный скролл
- FAQ-аккордеон
- Таймер до старта ретрита
- Лайтбокс галереи
- Модалка с формой + валидация
- CTA на Telegram `https://t.me/vsemaya`
- SEO/OG метатеги

## Перед публикацией
- Подставить точный адрес базы «Вкус жизни»
- Заменить `privacy.html` и `offer.html` на финальные юридические версии
- Вставить коды Яндекс.Метрики / Google Analytics
- При необходимости заменить изображения на финальные фото площадки и практик
- Конвертировать изображения в WebP (в текущем окружении отсутствует доступный WebP-конвертер)
